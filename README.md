# ScholarTrack — Scholarship Discovery & Application Tracker

A responsive scholarship discovery and application-tracking website built with
HTML, CSS, and JavaScript.

## Features
- Responsive navigation with active-link highlighting and sticky header
- Image slider, dashboard stats with animated counters, progress bars
- Drag & Drop shortlist demo, Local Storage & Session Storage demo
- Student registration form with JavaScript validation
- Light/Dark theme switcher
- Experiment 5: Advanced CSS — Grid, pseudo-elements, transforms,
  positioning, hover effects, CSS-only progress bars and tooltips
  (inline, internal, and external stylesheets all demonstrated)

## Tech Stack
HTML5, CSS3 (Grid, Flexbox, Custom Properties, Animations), JavaScript (DOM, Web Storage, Drag & Drop API), Font Awesome
